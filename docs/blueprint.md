# **App Name**: Fraud Detection System

## Core Features:

- Transaction Simulation: Input form to simulate credit card transactions with fields such as amount, time, location, and merchant details. Currency should be in INR (Rupees).
- Fraud Prediction: Real-time fraud prediction using a pre-trained machine learning model. Display prediction results (fraudulent or not) and confidence scores.
- Transaction History: Dashboard to view a history of simulated transactions, including their details and fraud statuses.
- Data Analysis Tool: Perform moderate data analysis on the simulated transactions.
- Real Prediction: It should have real prediction.
- Login System: Should have a login system.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey trust and security.
- Background color: Light blue (#E3F2FD) for a clean and trustworthy feel.
- Accent color: Soft green (#81C784) for positive confirmations, and deep red (#D32F2F) for potential fraud alerts.
- Body and headline font: 'Inter', a grotesque-style sans-serif known for its neutral and modern appearance, is recommended for both headlines and body text, because the app features a small amount of short text.
- Use simple, clear icons to represent transaction types and statuses.
- Clean, intuitive layout with clear separation of sections for input form, prediction results, and transaction history.
- Subtle animations to highlight fraud predictions and transaction updates.
